\# SuttonStone



Unityで制作中のパズルゲームです。  

対戦ズーキーパー風のゲームを目標に開発しています。



---



\## 🎮 ゲーム概要

\- ジャンル：パズルゲーム

\- プラットフォーム：PC

\- 開発環境：Unity

\- 制作人数：1人



---



\## 🧱 ゲーム内容

\- 石（Stone）を操作して盤面上で動かす

\- 一定条件を満たすと消去・スコア獲得

\- 今後、対戦要素やせり上がり要素を追加予定



---



\## 🛠 使用技術

\- Unity

\- C#

\- Git / GitHub



---



\## 📁 ディレクトリ構成

Assets/
├─ Scripts/ ゲームロジック
├─ Scenes/ シーン
└─ Prefabs/ プレハブ
Packages/
ProjectSettings/


---

## 🚀 実行方法
1. Unity Hubで本プロジェクトを開く
2. `Assets/Scenes` 内のシーンを開く
3. 再生ボタンを押す

---

## 📌 開発メモ
- Unity自動生成フォルダ（Library 等）は `.gitignore` により除外
- main ブランチで開発中
